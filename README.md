# Random-Quote-Generator
Генератор цитат — интерактивное приложение с графическим интерфейсом на Python, позволяющее случайным образом выбирать мудрые высказывания из предопределённого списка, просматривать и фильтровать историю сгенерированных цитат, а также сохранять эту историю между запусками программы в формате JSON.
