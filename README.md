## Описание программы

*Quote Generator — это приложение с графическим интерфейсом для генерации случайных цитат, их фильтрации по автору и теме, а также сохранения истории сгенерированных цитат.*

---

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.6 или выше

(Используются только стандартные библиотеки Python: `tkinter`, `random`, `json`, `os` — дополнительная установка не требуется)
